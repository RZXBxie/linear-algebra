# 矩阵公式速查表

> 学习日期：2026/06/09
> 用途：常用矩阵公式集中整理，方便随时查阅。约定：$A, B$ 为 $n$ 阶方阵（除非另有说明），$\lambda$ 为常数，$k$ 为正整数，$E$ 为单位矩阵。

---

## 一、运算律

### 加法 / 数乘

| 公式 | 说明 |
| --- | --- |
| $A + B = B + A$ | 交换律 |
| $(A + B) + C = A + (B + C)$ | 结合律 |
| $\lambda(A + B) = \lambda A + \lambda B$ | 数乘对加法分配 |
| $(\lambda + \mu) A = \lambda A + \mu A$ | 加法对数乘分配 |
| $(\lambda \mu) A = \lambda (\mu A)$ | 数乘结合律 |

### 矩阵乘法

| 公式 | 说明 |
| --- | --- |
| $(AB)C = A(BC)$ | 结合律 |
| $A(B + C) = AB + AC$ | 左分配 |
| $(B + C)A = BA + CA$ | 右分配 |
| $(\lambda A)B = A(\lambda B) = \lambda(AB)$ | 数乘可移 |
| $E_m A_{m\times n} = A_{m\times n} E_n = A$ | 单位元 |
| $AB \neq BA$ | ⚠️ 一般不交换 |
| $AB = O \;\not\Rightarrow\; A=O$ 或 $B=O$ | ⚠️ 无零因子律 |
| $AB = AC \;\not\Rightarrow\; B = C$ | ⚠️ 无消去律 |

---

## 二、转置 $A^T$

| 公式 |
| --- |
| $(A^T)^T = A$ |
| $(A + B)^T = A^T + B^T$ |
| $(\lambda A)^T = \lambda A^T$ |
| $(AB)^T = B^T A^T$ ← **顺序颠倒** |
| $(A_1 A_2 \cdots A_k)^T = A_k^T \cdots A_2^T A_1^T$ |
| $\lvert A^T \rvert = \lvert A \rvert$ |

**对称 / 反对称：** $A^T = A$ ⇔ 对称； $A^T = -A$ ⇔ 反对称。
- $A + A^T$ 对称，$A - A^T$ 反对称
- 任意方阵 $A = \tfrac12(A + A^T) + \tfrac12(A - A^T)$

---

## 三、方阵的幂

| 公式 |
| --- |
| $A^m A^n = A^{m+n}$ |
| $(A^m)^n = A^{mn}$ |
| $A^0 = E$ |
| $(AB)^k \neq A^k B^k$（一般情形） |
| $AB = BA \;\Rightarrow\; (AB)^k = A^k B^k$ |
| $AB = BA \;\Rightarrow\; (A+B)^n = \sum_{k=0}^{n} \binom{n}{k} A^k B^{n-k}$（二项式） |

---

## 四、行列式 $\lvert A \rvert$

| 公式 |
| --- |
| $\lvert A^T \rvert = \lvert A \rvert$ |
| $\lvert \lambda A \rvert = \lambda^n \lvert A \rvert$ |
| $\lvert AB \rvert = \lvert A \rvert \cdot \lvert B \rvert$ |
| $\lvert A^k \rvert = \lvert A \rvert^k$ |
| $\lvert A^{-1} \rvert = \lvert A \rvert^{-1}$ |
| $\lvert A^* \rvert = \lvert A \rvert^{n-1}$ |
| $A$ 可逆 $\iff \lvert A \rvert \neq 0$ |

---

## 五、逆矩阵 $A^{-1}$

| 公式 |
| --- |
| $A A^{-1} = A^{-1} A = E$ |
| $(A^{-1})^{-1} = A$ |
| $(\lambda A)^{-1} = \dfrac{1}{\lambda} A^{-1}$（$\lambda \neq 0$） |
| $(AB)^{-1} = B^{-1} A^{-1}$ ← **顺序颠倒** |
| $(A_1 A_2 \cdots A_k)^{-1} = A_k^{-1} \cdots A_2^{-1} A_1^{-1}$ |
| $(A^T)^{-1} = (A^{-1})^T$ |
| $(A^k)^{-1} = (A^{-1})^k$ |
| $A^{-1} = \dfrac{1}{\lvert A \rvert} A^*$ |

---

## 六、伴随矩阵 $A^*$

| 公式 |
| --- |
| $A A^* = A^* A = \lvert A \rvert \, E$ |
| $\lvert A^* \rvert = \lvert A \rvert^{n-1}$ |
| $(A^*)^{-1} = (A^{-1})^* = \dfrac{1}{\lvert A \rvert} A$（$\lvert A \rvert \neq 0$） |
| $(A^*)^T = (A^T)^*$ |
| $(\lambda A)^* = \lambda^{n-1} A^*$ |
| $(AB)^* = B^* A^*$ ← **顺序颠倒** |
| $(A^*)^* = \lvert A \rvert^{n-2} A$（$n \geq 2$） |

---

## 七、分块矩阵

设分块矩阵的子块在能进行运算的前提下：

| 公式 | 说明 |
| --- | --- |
| $\begin{pmatrix} A & O \\ O & B \end{pmatrix}^{-1} = \begin{pmatrix} A^{-1} & O \\ O & B^{-1} \end{pmatrix}$ | 准对角块求逆 |
| $\begin{pmatrix} O & A \\ B & O \end{pmatrix}^{-1} = \begin{pmatrix} O & B^{-1} \\ A^{-1} & O \end{pmatrix}$ | 反对角块求逆 |
| $\left\lvert \begin{matrix} A & O \\ C & B \end{matrix} \right\rvert = \lvert A \rvert \cdot \lvert B \rvert$ | 三角块行列式 |
| $\left\lvert \begin{matrix} O & A \\ B & O \end{matrix} \right\rvert = (-1)^{mn} \lvert A \rvert \lvert B \rvert$ | $A$ 为 $m$ 阶，$B$ 为 $n$ 阶 |

---

## 八、初等矩阵

由单位矩阵 $E$ 经一次初等变换得到的矩阵：

| 类型 | 记号 | $\lvert \cdot \rvert$ | 逆矩阵 |
| --- | --- | --- | --- |
| 交换 $i,j$ 行(列) | $E(i,j)$ | $-1$ | $E(i,j)$ |
| 第 $i$ 行(列) 乘 $k \neq 0$ | $E(i(k))$ | $k$ | $E(i(\tfrac{1}{k}))$ |
| 第 $j$ 行加 $k$ 倍第 $i$ 行 | $E(i,j(k))$ | $1$ | $E(i,j(-k))$ |

**关键性质：**
- 左乘 $E_{m}$：对 $A$ 作**行**变换；右乘 $E_n$：对 $A$ 作**列**变换
- 任何可逆矩阵均可表示为有限个初等矩阵的乘积

---

## 九、矩阵的秩 $r(A)$

| 公式 |
| --- |
| $0 \leq r(A) \leq \min(m, n)$ |
| $r(A^T) = r(A)$ |
| $r(kA) = r(A)$（$k \neq 0$） |
| $r(A + B) \leq r(A) + r(B)$ |
| $r(AB) \leq \min\{r(A), r(B)\}$ |
| $A$ 可逆 $\Rightarrow r(AB) = r(B),\; r(BA) = r(B)$ |
| $A_{m\times n} B_{n\times l} = O \;\Rightarrow\; r(A) + r(B) \leq n$（Sylvester 不等式） |
| $r\!\begin{pmatrix} A & O \\ O & B \end{pmatrix} = r(A) + r(B)$ |
| $A_{n}$ 可逆 $\iff r(A) = n$ |
| $r(A^*) = \begin{cases} n & r(A)=n \\ 1 & r(A)=n-1 \\ 0 & r(A) < n-1 \end{cases}$ |

---

## 十、常用结论速记

- **顺序颠倒三兄弟：** $(AB)^T = B^T A^T$、$(AB)^{-1} = B^{-1} A^{-1}$、$(AB)^* = B^* A^*$
- **行列式三连：** $\lvert AB \rvert = \lvert A \rvert \lvert B \rvert$、$\lvert \lambda A \rvert = \lambda^n \lvert A \rvert$、$\lvert A^* \rvert = \lvert A \rvert^{n-1}$
- **伴随核心：** $A A^* = \lvert A \rvert E$（一切伴随公式之源）
- **可逆等价命题：** $\lvert A \rvert \neq 0 \iff r(A)=n \iff Ax=0$ 仅零解 $\iff A$ 的行(列)向量线性无关 $\iff A$ 可由初等矩阵相乘得到
